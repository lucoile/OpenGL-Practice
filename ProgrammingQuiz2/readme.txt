Controls
***************************************************

 Esc -- exit
 
 R -- project camera 
	Arrow keys -- rotate project camera around origin
 M -- bodel front view camera
 B -- model back view camera
 S -- light camera

 L -- toggle lighting and shadows

 Q -- switch skybox between cube and sphere

 Shift + WASD -- move model forward, back, left, right
 A, D -- rotate selected model on x-z plane
 U, J -- increase and decrease selected model size

 References:
    https://learnopengl.com -- camera, mesh, model, shader classes and shadow algorithm
